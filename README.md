# Sequence Multiplayer Game

A full-stack multiplayer implementation of the Sequence board game with React frontend and Python Flask backend.

## Features

- ✅ **2-12 Players Support**: Play with 2 to 12 players simultaneously
- ✅ **Team Mode**: Optional 2-3 team gameplay
- ✅ **Real-time Multiplayer**: WebSocket-based real-time gameplay
- ✅ **Full Sequence Rules**: 
  - Standard 10x10 game board
  - Two 52-card decks
  - Two-eyed Jacks (♥♦) - place chip anywhere
  - One-eyed Jacks (♠♣) - remove opponent chip
  - Corner wildcards
  - Win with 2 sequences (5 in a row)
- ✅ **Room System**: Create/join rooms with shareable links
- ✅ **Admin Controls**: Room creator controls game start and settings
